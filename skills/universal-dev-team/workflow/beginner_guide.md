# 小白开发指南 (Beginner's Guide)

恭喜你迈出了编程的第一步！这个指南将帮助你理解我们将如何一起工作。

## 我们的合作模式

1.  **你只需要动嘴（或打字）**：告诉我们你想做什么，剩下的交给我们。
2.  **循序渐进**：我们不会一下子扔给你一堆复杂的代码。我们会一步一步来，确保你理解每一步。
3.  **随时提问**：不懂就问，架构师和开发者会用最通俗的语言给你解释。

## 常用术语表

*   **Repo (Repository)**: 仓库，存放代码的地方。
*   **PRD**: 产品需求文档，说明产品要长什么样。
*   **Frontend (前端)**: 用户看得到的界面。
*   **Backend (后端)**: 处理数据和逻辑的幕后程序。
*   **Deploy (部署)**: 把程序发布到网上，让别人能访问。
*   **Bug**: 程序里的错误。

## 准备工作

在开始之前，请确保你已经安装了以下软件（如果没有，开发者会教你安装）：
- VS Code (代码编辑器)
- Node.js (运行环境)
- Git (版本控制工具)

准备好了吗？让我们回到 `SKILL.md`，开始你的第一个项目吧！
